## Keymaps

### Editing

`gcc` Comment line

### Search and replace

`<leader>/` Grep search

`gw` Search word under cursor

`<leader><space>` Find in files (root dir)

### Windows and buffers

`<c-/>` Show terminal (root dir)

`<leader>|` Split window right

`<leader>-` Split window below
